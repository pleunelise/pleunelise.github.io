Project 1 HTML CSS In het eerste leerjaar begonnen we met HTML CSS. Hierbij was ook een project aan gelinkt, de opdrachtgever van dit project was Pasta Vino. We moesten in een groepje van 5 een HTML CSS pagina bouwen en daarbij ons natuurlijk aan de opdracht houden. We begonnen om elk voor zich een wireframe te maken die we daarna vergelijkt hebben waar ons eind wireframe uit kwam. Toen begonnen we allemaal aan een pagina te werken, de taak verdeling hadden we gemaakt in een trello board. Mijn taak was de pagina over de geschiedenis en bij ons eten. Zelf heb ik niet aan de CSS gewerkt, dat was iemand anders zijn taak.

Project 2 3D animaties in Blender In periode twee hadden we ons tweede project als aanleiding na een module. Bij dit project hebben we in opdracht van SOS kinderdorpen. De opdracht was in het thema blenderspeakssyria. We konden uit verschillende onderwerpen kiezen waaar we onze animatie over wouden houden. Na een klein overleg waren we eruit. We wouden het houden over 'Geen onderdak'. Dit onderwerp sprak ons erg aan. De teekverdeling ging erg goed, alleen de deadlines gingen war mindere omdat we daar nog niet echt gewend aan waren.

Project 3 Data visualisatie in Python In de derde periode kwam mijn favoriete project van het eerste school jaar. Het project ging over data visualisatie in python. Voor de aftrap van dit project zijn we naar Ortec geweest. Aangekomen bij Ortec werd de opdacht uitgelegd. We moesten allerlei verschillende grafieken maken over de fietsen diefstallen in Rotterdam. We werkten in Pycharm edu. Bij dit project gingen alle deadlines goed en waren er geen problemen. De taak verdeling ging ook heel gelijk. Met dit project heb ik veel leuke en nieuwe dingen geleerd.
